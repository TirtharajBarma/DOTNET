LIBRARY MANAGEMENT SYSTEM  
Scenario‑Based Assignment  
(Strict Task Instructions)

==================================================
TASK 1: ABSTRACT CLASS & ABSTRACT METHODS
==================================================

What the student MUST create:

1. ONE abstract base class  
   - Name: `LibraryItem`

2. The abstract class MUST contain:
   - Exactly **3 properties**:
     • Title  
     • Author  
     • ItemID  

   - Exactly **2 abstract methods**:
     • Method to display item details  
     • Method to calculate late fee  

3. TWO concrete child classes:
   - Book  
   - Magazine  

4. Each child class MUST:
   - Inherit from `LibraryItem`
   - Override **both** abstract methods

5. Late fee rules (MANDATORY):
   - Book → 1 unit per day
   - Magazine → 0.5 units per day

--------------------------------------------------
Proof Required:
--------------------------------------------------
- Create:
  • 1 Book object  
  • 1 Magazine object  

- Display:
  • Title  
  • Author  
  • ItemID  
  • Late fee for **exactly 3 days**

==================================================

What the student MUST create:

1. ONE interface:

2. ONE interface:
   - Name: `INotifiable`
   - Contains ONE method that accepts a message

3. The `Book` class MUST:
   - Implement both interfaces
   - Implement all interface methods

--------------------------------------------------
Proof Required:
--------------------------------------------------
- Call:
  • Reservation method  
  • Notification method  

- Output MUST clearly show:
  • Reservation success  
  • Notification message sent
